# Figure Storage Directory

This directory (`man/figures/`) stores static PNG snapshots of rgl 3D visualizations generated by vignettes and pkgdown articles.

## Purpose

- **pkgdown compatibility**: pkgdown automatically finds figures in this location
- **Vignette support**: Vignettes reference these figures via relative paths
- **Version control**: Figures are tracked in git for reproducibility

## How Figures Are Created

Figures are generated programmatically using the `save_rgl_snapshot()` function from `R/rgl_utils.R`:

```r
library(rgl)
library(houghPrimitives)

# Create 3D scene
open3d()
spheres3d(0, 0, 0, radius = 1, color = "blue")

# Save to man/figures/
fig_path <- save_rgl_snapshot("my-figure.png")

close3d()
```

## Naming Convention

Use descriptive, hyphenated names:

- ✓ `sphere-detection-example.png`
- ✓ `line-detection-comparison.png`
- ✓ `multi-shape-results-3d.png`
- ✗ `fig1.png` (too generic)
- ✗ `temp.png` (unclear purpose)

## File Management

- **Generated files**: Created by `save_rgl_snapshot()` during vignette builds
- **Not manual**: Do NOT place manually-created screenshots here
- **Reproducible**: All figures can be regenerated by knitting vignettes
