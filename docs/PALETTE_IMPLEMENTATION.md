## Functions Implemented

### 1\. hough\_palette(n, type = c(“full”, “contrast”))

**Purpose:** Primary palette function for getting n colors

**Features:** - Returns n colors from specified palette - Automatically
cycles if n \> palette size using `rep_len()` - Two palette types: -
`"full"` - 20 colors (dark to bright, sequential) - `"contrast"` - 19
colors (high contrast, categorical)

**Source palettes:**

``` r
# Full palette (20 colors)
c("#0B0E14", "#1F2937", "#2E3440", "#FF9F1C", "#FF7A18", "#E4572E", 
  "#FF3C38", "#FFD166", "#FCA311", "#E9C46A", "#84BC48", "#2EC4B6",
  "#00BBF9", "#4CC9F0", "#4895EF", "#560BAD", "#9D4EDD", "#C77DFF",
  "#B5179E", "#F72585")

# Contrast palette (19 colors)
c("#FF9F1C", "#4CC9F0", "#E4572E", "#2EC4B6", "#FFD166", "#7209B7",
  "#FF3C38", "#00B4D8", "#E9C46A", "#B5179E", "#4895EF", "#F4D35E",
  "#9D4EDD", "#A8DADC", "#1F2937", "#EAEAEA", "#2E3440", "#FFFFFF",
  "#0B0E14")
```

### 2\. hough\_palette\_gradient(n, type, alpha = 1.0)

**Purpose:** Create smooth color gradients via interpolation

**Features:** - Uses `colorRampPalette()` for seamless transitions -
Supports transparency via alpha parameter (0.0-1.0) - Skips very dark
colors (\#0B0E14, \#1F2937, \#2E3440) to maintain visibility - Ideal for
continuous data (elevation, intensity)

**Implementation:**

``` r
# Extract colors for interpolation (skip dark anchors)
base_colors <- if (type == "full") {
  FULL_PALETTE[4:20]  # Skip first 3 dark colors
} else {
  CONTRAST_PALETTE[1:14]  # Skip last 5 (includes dark colors)
}

# Create gradient with transparency
colorRampPalette(base_colors, alpha = TRUE)(n)
# Then apply alpha if specified
```

### 3\. hough\_palette\_pairs(n)

**Purpose:** Get high-contrast foreground/background color pairs

**Features:** - Returns 2\*n colors (n pairs) - 10 predefined
contrasting pairs - Optimized for text readability and label visibility
- Automatically cycles if n \> 10

**Predefined pairs:**

``` r
pairs <- c(
  "#FF9F1C", "#0B0E14",  # Orange on dark
  "#4CC9F0", "#1F2937",  # Cyan on dark gray
  "#E4572E", "#FFFFFF",  # Red-orange on white
  "#2EC4B6", "#2E3440",  # Turquoise on gray
  "#FFD166", "#7209B7",  # Yellow on purple
  "#FF3C38", "#0B0E14",  # Red on dark
  "#00B4D8", "#FFFFFF",  # Deep cyan on white
  "#E9C46A", "#1F2937",  # Soft yellow on dark
  "#B5179E", "#EAEAEA",  # Magenta on light gray
  "#4895EF", "#2E3440"   # Blue on gray
)
```

### 4\. hough\_palette\_preview(n, type)

**Purpose:** Visualize palette colors via barplot

**Features:** - Displays barplot with colored bars - Each bar shows a
color from the palette - Useful for exploring color combinations - Shows
hex codes on bars

**Implementation:**

``` r
colors <- hough_palette(n, type)
barplot(rep(1, n), col = colors, border = NA,
        main = sprintf("houghPrimitives %s Palette (%d colors)", 
                      toupper(type), n),
        xlab = "Color Index", ylab = "",
        names.arg = 1:n)
```

## Use Case Examples

### Categorical Data (Detected Shapes)

``` r
result <- detect_spheres(points, label_points = TRUE)
colors <- hough_palette(result$n_spheres, "contrast")
# Each sphere gets a distinct, high-contrast color
```

### Continuous Data (Elevation Gradient)

``` r
elevation_colors <- hough_palette_gradient(100, "full")
elevation_bins <- cut(points[,3], breaks = 100)
point_colors <- elevation_colors[elevation_bins]
# Smooth color transition from low to high elevation
```

### Classification

``` r
class_colors <- hough_palette(7, "contrast")
las$RGB <- class_colors[las$Classification]
plot(las, color = "RGB")
# 7 distinct colors for classification classes
```

### Text Labels

``` r
pairs <- hough_palette_pairs(5)
# pairs[1] = foreground, pairs[2] = background
# pairs[3] = foreground, pairs[4] = background, etc.
# High contrast for readable text/labels
```

## Design Principles

### Full Palette (20 colors)

  - **Sequential ordering**: Dark → Medium → Bright → Saturated
  - **Color progression**: Blues → Oranges → Yellows → Greens → Cyans →
    Purples
  - **Use cases**: Elevation, intensity, distance, time series

### Contrast Palette (19 colors)

  - **Perceptual separation**: Maximum visual distinctness
  - **Color distribution**: Spread across hue spectrum
  - **Use cases**: Categories, shapes, instances, classifications

### Gradient Interpolation

  - **Smooth transitions**: No jarring color jumps
  - **Visibility**: Skips very dark colors for better contrast
  - **Flexibility**: Unlimited colors via interpolation

### Contrasting Pairs

  - **High luminance contrast**: Dark backgrounds with bright
    foregrounds
  - **Readability**: Optimized for text/labels
  - **Accessibility**: Considers color blindness (varies hue and
    luminance)

## Integration with lidR

Seamless integration demonstrated:

``` r
library(houghPrimitives)
library(lidR)

# Detect shapes
result <- detect_spheres(points, label_points = TRUE)

# Assign colors
colors <- hough_palette(result$n_spheres, "contrast")

# Create LAS
las <- LAS(data.frame(
  X = points[,1], Y = points[,2], Z = points[,3],
  SphereID = result$point_labels
))

# Plot with automatic coloring
plot(las, color = "SphereID", size = 3)

# Or manually assign
las$RGB <- colors[result$point_labels]
plot(las, color = "RGB", size = 3)
```
