# houghPrimitives: 3D Shape Detection in Point Clouds Using Hough Transforms

Detect geometric primitives (lines, planes, spheres, cylinders) in 3D
point cloud data using Hough transform algorithms.

## Details

This package provides tools for detecting geometric shapes in 3D point
clouds, commonly used in LiDAR and photogrammetry applications.

The main function currently available is `summarize_points()`, which
provides basic statistics about point cloud data.
